# Notice

This repository is a sanitized portfolio excerpt of a Flutter mobile
application. It is published to demonstrate code organization, architecture,
state management, and mobile implementation patterns.

## Sanitized Configuration

The following production-specific values were intentionally removed or replaced
with placeholders:

- Backend API base URL
- Firebase project IDs, API keys, app IDs, sender IDs, and client IDs
- Android application ID
- iOS bundle ID
- Local build output and IDE-generated state

The placeholder configuration is suitable for code review, but it is not
intended to connect to any production service.

## Running The Project

The project targets an older Flutter/Dart SDK range:

```yaml
sdk: ">=2.17.5 <3.3.3"
```

Use a compatible Flutter SDK and provide your own backend and Firebase
configuration before attempting to run the app.

## Scope

This repository should be treated as a technical excerpt, not a complete
open-source product distribution. Screens, flows, and integrations may depend on
services and operational context that are not included in this public version.
